## 📞 SUPPORT & TROUBLESHOOTING

### Common Issues & Solutions:

#### Frontend Issues:
- **Can't compile:** Check [COMPILE_AND_RUN.md](COMPILE_AND_RUN.md)
- **JavaFX not found:** Install JavaFX SDK
- **Game not starting:** Check Maven dependencies

#### Backend Issues:
- **Port 8080 in use:** Change server.port in application.properties
- **JWT errors:** Check jwt.secret configuration
- **WebSocket won't connect:** Check CORS settings
- **H2 console empty:** Verify JPA ddl-auto=update
